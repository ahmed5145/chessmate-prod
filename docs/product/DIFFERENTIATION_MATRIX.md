# Differentiation Matrix

## Summary

Chess.com and Lichess are strong at single-game engine analysis. ChessMate should win by turning many games into one clear coaching diagnosis.

| Capability | Chess.com | Lichess | ChessMate MVP |
| --- | --- | --- | --- |
| Single-game engine lines | Strong | Strong | Present, but not the main value |
| Batch analysis across many games | Limited | Limited | Core feature |
| Recurring weakness detection | Limited | Limited | Core feature |
| Action plan after analysis | Basic | Basic | Core feature |
| Coach-style explanation | Moderate | Moderate | Core feature |
| Progress over time across batches | Limited | Limited | Planned |
| Personalized next-step guidance | Limited | Limited | Core feature |
| Desktop-first coaching workflow | Moderate | Moderate | Core focus |
| Emailed completion for long jobs | Rare | Rare | Planned |
| Value from 20-50 games at once | Low | Low | High |

## What ChessMate Should Not Compete On First

- Best engine line viewer.
- Opening database depth.
- Free-analysis breadth.
- Social features.

## What ChessMate Should Own

- "Here are the 2-3 things holding you back most."
- "Here is the pattern across your games."
- "Here is what to do next."
- "Here is whether you improved after the last batch."

## Positioning Statement

ChessMate is a private coach for players who already play enough games to have a pattern, but do not want to manually discover that pattern themselves.
